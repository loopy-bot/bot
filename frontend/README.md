# Admin 架构

## 详细文档

http://adminpro-doc.mt178.com/

## 介绍
1. vue3 后台管理系统，基于vite3创建
2. 设计基于arco design
3. 支持多种布局、主题，可自定义
4. 支持低代码表单
5. 自动采集生产日志
